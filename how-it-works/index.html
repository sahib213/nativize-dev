How it works

From Lovable to app stores in three steps.

Nativize keeps generation in your browser, writes a real Capacitor project, and lets GitHub Actions compile the native app artifacts in your own repository.

The clean path

No random export maze. Open your Lovable project, choose native options, then generate and push the kit.

1

Detect

Open your Lovable project and let Nativize detect the app name, build output, and GitHub repo.

2

Generate

Choose app permissions, push, social sign-in, signing, and store upload options. Nativize writes the native kit.

3

Build

Push to GitHub and run workflows for installable iOS, Android, Mac, and Windows artifacts.
