Nativize

Lovable apps → native iOS, Android, Mac + Windows

1Open your project on lovable.dev.
2Click the floating Nativize button at the bottom-right.
3Confirm the details, then Download .zip or Push to GitHub.
4Run Actions → Nativize Build to build in the cloud — free, no Xcode needed.
If Chrome opened this page from the toolbar icon, the current tab blocks extensions. Open a normal web page or Lovable project, then click Nativize again.
