Security

Your code and keys stay under your control.

Nativize uses browser-local generation and GitHub-owned builds. Support forms only send the text you submit.

Local by default

The generator runs in your browser. Nativize does not need to upload your source code to its own backend.

Token stored locally

GitHub tokens stay in browser storage and are sent to Supabase only for secure download checks. Store credentials become encrypted GitHub secrets.

You own the output

The kit is plain Capacitor in your own repo, so you can inspect it, edit it, and keep it.
